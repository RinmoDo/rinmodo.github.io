Wireless Vibration Sensor

This project was my final year study project, in more than 6 months, and in a team over 20 members, we were able to give birth to an industrial sensor in the international market; this sensor can monitor temperature and vibration wirelessly with an advanced Deep learning module that can analyse & predict the next 6months failure with an accuracy of 86%. This system allows to company monitor their machines anywhere from a visual dashboard via an API; and predict the failure to react quickly...

Embedded facial access system

We developed a facial recognition access control device; all the system is embedded in a tiny box (26x32mm) attached to a 3D printed electrical door lock created. This facial access control device, associated with the latest advances in Artificial Intelligence and facial recognition algorithms, and linked to a Telegram BOT that notify the user and give him control over all.
